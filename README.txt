== Description ==

Powerful and reliable SMTP delivery service that enables you to send and track high-volume emails effortlessly.

== Installation ==

1. Upload `smtp-com-mail` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
